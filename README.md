# OHASIS Desktop Releases

Welcome to the official release repository for **OHASIS (One Health AMR Surveillance and Information System)**.

This repository provides downloadable installers for the OHASIS Desktop application.

---

## 📥 How to Download & Install

### 🪟 For Windows

1. Go to the **Releases** section of this repository
2. Download the latest `.exe` file
3. Double-click the file to start installation
4. Follow the on-screen instructions
5. Once installed, open **OHASIS Desktop** from your Start Menu

---

### 🍎 For macOS

1. Download the `.dmg` file from the **Releases** section
2. Open the downloaded file
3. Drag the OHASIS app into the **Applications** folder
4. Open the app from Applications

---

### 🐧 For Linux

1. Download the `.AppImage` or `.deb` file
2. For `.AppImage`:

   * Right-click → Properties → Make it executable
   * Double-click to run
3. For `.deb`:

   * Open with your package installer and install

---

## 🔄 Updates

* The application will notify you when a new version is available
* You can update directly from the app

---

## ⚠️ Important Notes

* Always download from the official **Releases** section
* Do not install files from unknown sources
* If you see a security warning, choose **“More info → Run anyway”** (Windows) if you trust the source

---
